# Getting Started
[Demo](https://lineage-main.vercel.app/)
## Prerequisites

- Node 19 (v19.1.0)

## Available Scripts
- `yarn` - Install all node dependencies for a project
- `yarn start` - Start the development server
- `yarn build` - Bundles the app into static files for production
